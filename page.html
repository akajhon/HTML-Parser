HTML-Parser

Este projeto usa Python para analisar arquivos HTML e extrair informações sobre as tags <script>, <link> e <img>.

São exibidos o tamanho dos arquivo encontrados e, para códigos, uma pré visualização das 5 primeiras linhas em cada arquivo.

Uso

  1. Instale o Python 3+
  2. Baixe o código do HTML-Parser
  3. Execute o comando python main.py e especifique o arquivo HTML que deseja analisar
  4. Será exibido informações sobre as tags encontradas no arquivo e as imagens encontradas serão abertas no visualizador padrão de imagens do sistema operacional.

Abaixo estão inseridas algumas tags <script>, <link> e <img>. Você pode executar "python main.py" e escolher o arquivo "pagina.html" para visualizar as informações.
